# MyReads Project

This is the first project assignment of the [**React** Udacity's Nano degree program](https://www.udacity.com/course/react-nanodegree--nd019

# Installation & Running

## To install it:

At the project's root directory :

`yarn install`

## To run it:

`yarn start`
